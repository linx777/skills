# template-skill

Provides a starter template for building new skills.

## Overview

Offers a minimal skill scaffold with SKILL.md, metadata placeholders, and example triggers to speed up authoring.

## Setup

Command:

```bash
claude mcp add template-skill
```

YAML config:

```yaml
tools:
  - name: template-skill
    template: base
    files: starter
```
