# brand-guidelines

Applies Anthropic brand colors and typography across decks, docs, and artifacts.

## Overview

Keeps artifacts on-brand by enforcing Anthropic colors, Poppins/Lora typography, and accent cycling. Great for post-processing slides, docs, and visual assets that need the official Anthropic look without manual styling.

## Features

- Applies Anthropic palette (dark, light, mid gray, accent colors)
- Uses Poppins for headings and Lora for body with smart fallbacks
- Cycles accent colors on non-text shapes while preserving hierarchy
