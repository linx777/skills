# algorithmic-art

Builds seeded, interactive p5.js generative art from an algorithmic philosophy manifesto.

## Overview

A creative coding skill that first writes an algorithmic art philosophy, then ships a self-contained p5.js artifact with seed controls and parameter sliders. It emphasizes emergent systems, controlled randomness, and master-level craft for gallery-ready generative works.
