# theme-factory

Applies curated color/font themes or generates new ones for any artifact.

## Overview

A theming toolkit with 10 preset palettes and font pairings that can be applied to slides, docs, HTML, or generated on the fly. Ensures cohesive color systems and typography across artifacts.

Command:

```bash

```

YAML config:

```yaml
tools:
  - name: theme-factory
    presets: 10
    generate: true
```
