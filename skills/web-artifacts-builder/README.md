# web-artifacts-builder

Builds complex claude.ai artifacts with React, Tailwind, and shadcn/ui.

## Overview

A frontend toolkit for multi-component claude.ai artifacts that need state management, routing, or shadcn/ui components. It scaffolds repos, applies Tailwind, and encourages production-grade React patterns.
