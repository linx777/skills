# internal-comms

Templates and guidance for internal updates, newsletters, and incident comms.

## Overview

Provides structured patterns for internal communication: 3P updates, leadership notes, FAQs, and incident reports. It tunes tone for audience, keeps brevity, and enforces clarity so updates land well.
