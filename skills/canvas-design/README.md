# canvas-design

Crafts design philosophies then renders museum-grade posters as PNG/PDF.

## Overview

A visual design skill that writes a design manifesto and then expresses it as a polished static artifact. It prioritizes spatial composition, minimal text, meticulous craft, and export-ready PNG/PDF outputs.
